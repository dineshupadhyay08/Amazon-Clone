
# Amazon Clone

A responsive Amazon clone website built using HTML, CSS, and Bootstrap, following the Apna College video tutorial.



## Features
- Responsive Design with Bootstrap
- Home Page with Product Listings
- Product Detail Pages
- Shopping Cart Functionality
- User Login & Registration Forms
 


## 🛠 Skills
HTML, CSS...


## Tech Stack

**Frontend:**  HTML, CSS, Bootstrap

**Version Control:**  Git & GitHub



## Installation

Install my-project with npm

```bash
  git clone https://github.com/your-username/amazon-clone.git cd amazon-clone

```
